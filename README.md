## Description

The Candidate Search Application is a web-based tool that allows users to search for GitHub users, save potential candidates for review, and view their details. This application leverages the GitHub API to fetch user data, providing a seamless way to explore and shortlist candidates.

- Search for candidates using GitHubâ€™s user data.
- Save potential candidates to a list.
- View saved candidates with detailed information, including:
  - Name
  - Username
  - Location
  - Email
  - Company
  - GitHub profile link

## Technologies Used

- **React**: Frontend library for building the user interface.
- **TypeScript**: For strong typing and improved code quality.
- **GitHub API**: To fetch candidate data.
- **CSS**: For styling the application.
- **React Router**: For navigation between pages.
